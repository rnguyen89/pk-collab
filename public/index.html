About the App

This app is styled using Materialize.css framework. Essentionally, it is a a Kanban board for parents to use with there children. The flow is similar to using sticky notes, but there are points you can add to stickies. The value, is to have task in a point sytem where you can see progress and jot down a reward for all completed tasks, again, based on points or progress.

Visualize Rewards

It is always great to see a progress bar to show that goals or rewards are within reach. Having the progress or reward modal is for parents to collaborate with children about rewards.

Technology Stack

Frontend: HTML5, CSS5, JavaScript, jQuery, and Materialize.css.

Backend: Node.js + Express (web server), MongoDB (database), Mocha + Chai (testing), Continuous integration and deployment with Travis CL.

Security: JSON Web Tokens (user authentication).
